# Projeto de Interação Humana com o Computador
## O que foi feito?
Este projeto é uma recriação do jogo clássico Minesweeper (Campo Minado), tendo em foco o design de interação e a experiência do utilizador com uma implementação em JavaFX. A ideia foi não só replicar a lógica do jogo original, mas também melhor a usabilidade através de um interface intuitivo criado no SceneBuilder.

Funcionalidades implementadas:
- Posicionamento aleatório das bombas em cada nova partida, garantindo rejogabilidade e imprevisibilidade
- Duas interações distintas por clique: clique esquerdo para revelar bloco, clique direito para colocar ou retirar a bandeira (feedback visual imediato para ambas as ações)
- Três níveis de dificuldade com tabuleiros de tamanhos progressivamente maiores (fácil, médio, difícil), cada um com densidade adequada de bombas
- Sistema de numeração inteligente que indica o número exato de bombas adjacentes a cada célula revelada, seguindo a lógica clássica do Minesweeper
- Interface visual personalizada desenhada no SceneBuilder, com cores bem contrastadas e layout responsivo
